# get_score_data
譜面保管所にある譜面画像から譜面データに変換するツールです　　　　
4分、8分、16分、24分、32分は正しくデータ化できます　　
つまみは回す方向(左 or 右)でデータ化されます　　
譜面データはcsvファイルで保存されます　　
一部バグが残っているため、修正必須です　　
最新バージョンの譜面データが読み取れない可能性有　要検証

# 使い方
1.https://sdvx.in/にアクセスして、譜面データが欲しい楽曲の譜面画像htmのURLをコピーします　　
2.コピーしたいURLと曲名を入力して「譜面データ取得」を押す
3.「譜面データ読み取り完了!」というポップアップがでれば成功
